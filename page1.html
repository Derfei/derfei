Distributed Platform with Edge Server and Remote cloud
Project description:

This platform consist mobile device, edge server and remote cloud.
Mobile device can offload the computation of deep learning applications to edge server and remote cloud. The deep learning applications are represented as a DAG. Edge server and remote cloud provide a web service with Flask to get the tasks of DAG from mobile device. In the back-end of web service, the computation is implemented with Tensorflow.
